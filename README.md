# Carousel原生js写的轮播  
## Carousel使用  
引入css 和 js 文件:    
<pre>
 link rel="stylesheet" type="text/css" href="carousel.css   
 script type="text/javascript" src="carousel.js" 
</pre>    
  
  html基本结构: div\>(div\>div\>img或者a)+div\>左右切换按钮,id或者css选择器自定义.
